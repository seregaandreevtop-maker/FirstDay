# FirstDay

Нуарный HTML-трекер воздержания от вредных привычек.

## Возможности
- Ввод имени, даты рождения, привычки и даты старта воздержания.
- Онлайн-счётчик: **лет, дней, часов, минут, секунд** без срыва.
- Ежедневное мотивирующее напоминание (в интерфейсе и через Web Notification API, если разрешено).
- Подбор мотивационной цитаты под выбранную привычку.
- Кнопка **«Сбросить результат»** — начинает отсчёт заново с текущего момента.
- Все данные сохраняются локально в браузере (`localStorage`).

## Запуск
Откройте `index.html` в браузере.

## Публикация на GitHub
1. Загрузите файлы репозитория на GitHub.
2. Включите **GitHub Pages** в настройках репозитория:
   - Settings → Pages
   - Source: Deploy from a branch
   - Branch: `main` (root)
3. Приложение станет доступно по публичной ссылке GitHub Pages.
